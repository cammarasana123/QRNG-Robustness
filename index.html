Quantum Randomness Demo

An interactive browser demo inspired by your simulation study: generate a bit sequence, vary readout and gate noise, apply a simple privacy-amplification step, and watch real-time randomness indicators update. It is educational rather than cryptographically certified, but it mirrors the logic of the report in a GitHub Pages-friendly way.

Controls

Interpretation. The demo uses a synthetic “quantum-content” score that decreases with noise, plus practical sequence diagnostics: balance, runs, lag-1 correlation, byte entropy, and a visual randomness meter. It is meant for presentation and outreach on GitHub Pages.
